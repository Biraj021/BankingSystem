# Simple Banking System in C++

## Description
A simple banking system implemented in C++ using OOP concepts. 
Supports multiple accounts, deposits, withdrawals, and balance checking.

## Features
- Create multiple account objects
- Deposit money with validation
- Withdraw money with validation
- Check account balance

## Concepts Used
- Classes and Objects
- Constructors
- Encapsulation (private Balance)
- Member Functions
- Input Validation

## How to Run
1. Clone the repository
2. Open Account.cpp in a C++ IDE
3. Compile and run
